# 프로젝트 기획서 (PRD) - 마이링크(My Link)

## 1. 프로젝트 개요
- **프로젝트명**: 마이링크 (My Link)
- **목적**: 자신의 다양한 활동(포트폴리오, 블로그, SNS 등)을 하나의 링크로 모아서 공유할 수 있는 심플한 멀티 링크 프로필 서비스 제공.
- **대상 사용자**: 유튜브, 인스타그램, 틱톡 등 다양한 채널을 운영하며 단일 홍보 링크가 필요한 크리에이터 및 일반 사용자.
- **주요 기술 스택**: 
  - **프론트엔드**: Shadcn UI를 적극 활용한 모던 웹 환경 (React 생태계 권장)
  - **백엔드/인증**: Firebase (Authentication - 구글 소셜 로그인 전용)
  - **데이터베이스**: Firebase Firestore

---

## 2. 데이터베이스 모델링 가이드
- Firestore를 기반으로 작성되며 **서브 컬렉션(Subcollection)** 구조를 채택하여 데이터를 구성합니다. (예: `users` 컬렉션 하위에 `links` 서브 컬렉션 등)
- 사용자 프로필 정보 모델에는 두 가지 이름 필드가 반드시 포함되어야 합니다:
  - `displayName`: 프로필 닉네임 (URL 슬러그 생성 기준)
  - `username`: 실제 사용자의 이름 (본명 등)

---

## 3. 핵심 기능 목록

### 🎯 필수 기능
1. **회원가입 및 인증 (구글 소셜 로그인)**
2. **프로필 텍스트 정보 관리** (닉네임, 실제 이름, 한 줄 소개 - **인라인 편집**)
3. **고유 URL 생성** (별도의 핸들 없이 닉네임(`displayName`) 기반 URL 슬러그 설정)
4. **링크 아이템 추가 및 관리 (CRUD)** (추가, **인라인 편집**, 삭제 및 구글 API 파비콘 추출)
5. **프로필 퍼블릭 페이지 뷰** (사용자 지정 URL로 누구나 접속 가능한 뷰어 페이지)

### 📈 추후 추가 예정 기능
1. **링크 클릭 조회수 기능**: 각 링크 아이템별 클릭 수를 트래킹하고 보여주는 기능.

---

## 4. 각 기능의 상세 설명

### 4.1 회원가입 및 인증
- **설명**: Firebase Authentication을 사용하여 가입 및 로그인을 처리합니다.
- **상세 내용**:
  - 구글(Google) 소셜 로그인 단일 방식으로만 제공하여 가입 장벽을 최소화합니다.
  - 가입 시 사용자의 **구글 지메일(Gmail) 계정 주소 중 앞부분(`@` 이전의 ID 부분)**을 추출하여 `displayName`(닉네임)의 초기값으로 자동 설정합니다.

### 4.2 닉네임 기반 고유 URL 슬러그 설정
- **설명**: 사용자마다 고유한 접근 링크를 부여합니다.
- **상세 내용**:
  - 별도의 아이디(핸들)를 생성하지 않고, 사용자의 **닉네임(`displayName`)**을 URL 슬러그(Slug)로 사용하여 프로필 주소를 설정합니다. (예: `도메인.com/닉네임` 또는 `도메인.com/@닉네임`)
  - 해당 주소 접속 시 별도의 로그인 없이 누구나 프로필을 볼 수 있는 퍼블릭 라우팅을 처리합니다.

### 4.3 프로필 정보 관리 (인라인 편집)
- **설명**: 페이지 상단에 노출될 개인 브랜딩 영역(헤더) 텍스트를 관리합니다.
- **상세 내용**:
  - 닉네임(`displayName`), 실제 이름(`username`) 및 한 줄 소개(Bio) 텍스트를 자유롭게 수정할 수 있습니다.
  - 별도의 편집 페이지나 모달 팝업으로 이동하지 않고, 화면에서 텍스트를 클릭하여 바로 수정하는 **인라인 편집(Inline Editing)** 방식을 사용합니다.
  - *(참고: 프로필 이미지 업로드 기능 및 모바일 뷰 미리보기 기능은 제공하지 않습니다. 또한 커스텀 테마 기능은 제외됩니다.)*

### 4.4 링크 아이템 추가 및 관리 (CRUD)
- **설명**: 실제 방문자에게 보여줄 외부 서비스 링크들을 관리합니다.
- **상세 내용**:
  - **추가**: 링크 타이틀과 URL을 입력하여 새로운 버튼(블록)을 생성합니다.
  - **파비콘(Favicon) 자동 연동**: **구글 파비콘 API(Google Favicon API)**를 사용하여 입력한 대상 URL의 파비콘 이미지를 추출하고, 해당 링크 아이템의 좌측 아이콘으로 자동 설정합니다.
  - **인라인 수정**: 링크의 타이틀과 URL 역시 별도 창 없이 리스트에서 텍스트를 클릭해 바로 수정하는 **인라인 편집**을 지원합니다.
  - **삭제**: 기존 링크를 삭제할 수 있습니다.
  - *(참고: 드래그 앤 드롭을 통한 순서 변경, 링크 활성화/비활성화 토글 기능은 제공하지 않습니다.)*

### 4.5 추후 추가 예정 기능 (링크 클릭 조회수)
- **설명**: 등록된 링크들이 얼마나 클릭되었는지 성과를 파악합니다.
- **상세 내용**:
  - 방문자가 프로필 뷰어 페이지에서 각 링크 아이템을 클릭할 때마다 조회수를 카운트하고 기록하는 기능을 추후 구현합니다.
